Quick Intro to deck.js

How to Make a Deck

  1. Copy that Github repo

    git clone git@github.com:vzmind/personal-deckjs.git

  2. Customize the introduction.css if needed

    It's already woming with a basic shadowing style.

How to add slides

  1. Simply edit HTML

    Edit the index.html file

  2. A section tag is a slide

    Just add as much as you want.

  3. Choose Themes

    One for slide styles and one for deck transitions.
